# Destini 
This project aims to make a Destini story app, in which you have a story, depending on your choice each time you will get a certain path.
this project was part of https://github.com/londonappbrewery tutorials.

## App Layout:
This project consists of:  
1- one page.  
2- two buttons to select what you will do.  
3- story file.  

![Finished App](https://github.com/londonappbrewery/Images/blob/master/Destini.gif)

## Features:
    1- simple layout.
    2- easy to insert inputs.
    3- Story app, which I can edit to insert a great story.
    4- Based on user choice I can easily knows what is his/her personality looks like.
    5- works on both Android and IOS.
